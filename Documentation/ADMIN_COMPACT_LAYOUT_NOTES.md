# v4.0 Admin Compact Data Mode

This patch switches the admin area from presentation-sized tables to compact data tables.

## What changed

- Removed sticky desktop action columns.
- Reduced table cell font sizes and padding.
- Reduced button and badge sizes inside admin tables.
- Reduced product image thumbnails in the Digital Products manager.
- Stabilized filter bars so controls wrap cleanly.
- Kept responsive card layout for small screens.

## Why

The previous admin layout used large presentation spacing and sticky action columns. On desktop this caused action columns to overlay status/payment columns and made buttons appear missing or clipped.
